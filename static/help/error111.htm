ERROR 11:

          Could not read dependency file <File Name>

Description:

om was pass –g to create an Impact Analysis report on the Knowledge Base Server.  The om creates a depend.xml file and then sends it to the Knowledge Base Server.  The depend.xml file could not be read.

 

Resolution:

Make sure the build directory is writable and that a depend.xml file can be created.  You should also make sure that there is space available in the directory.
