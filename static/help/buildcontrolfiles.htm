Build Control Files

 

The OOMLBLDM program reads information from the Knowledgebase server and generates a file that contains information regarding the details of the build.  OOMLBLDM uses the Knowledgebase Rules information based on the build type specified by the Target file (TGT) for which the build is being executed, along with specific source code information specified by the TGT file and creates a file that controls how the build will be completed. This Build Control file is used as input to OOMLOM.
