Adding or Removing Users from Groups

 

In order to assign Users to Groups, you must have Administrator privileges.  Once the Administrator has created and customized a new Group, Users can be added to the Group.  Table GU-4 will get you started:

 

Table GU-4  Adding Users to a Group

Steps

Action

1

From the main menu select Windows – Open Perspective – OM Admin.

2

Select the Group Explorer tab for a list of all Groups – Double Click on the Group to which you want the user to be added.

3

View the Overview tab to view all Users – notice the Users with checks next to their name. These are the Users assigned to the group.

4.

To add a User to the Group, select the checkbox next to the User, and save the Group. To remove a User from the Group, uncheck the checkbox next to the User name and save the Group.
