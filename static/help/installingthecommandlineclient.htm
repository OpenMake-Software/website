Installing the Command Line Client

 

The command line client is installed automatically when the Knowledgebase Server is installed on the first launch of the Meister.  The required command line client will be installed in the client/bin directory relative to the dds.exe.  The command line client is always used to execute Workflows and is a necessary component of the overall Meister processing.

 

The PATH environment variable is not set by default to include the client/bin directory.  If you want to run the Command Line Client executables from a command line prompt then you will need to add <installation root>\client\bin to your PATH manually.
