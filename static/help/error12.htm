ERROR 12:

          Couldn't open the Build Control File: < Build Control File name>

Description:

When om began to read the Build Control File, it could not find the *.mak file in the local or specified directory.

Resolution:

Rerun bldmake or make sure the Build Control File is not read-only. Also if the -c parameter is used to change the working directory, make sure the Build Control File (*.mak) is in the directory being changed to.
