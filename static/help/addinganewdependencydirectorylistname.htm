Adding a New Dependency Directory List Name

Lists of Directories are named to simplify the reference of the complete list. Existing Projects can have new Dependency Directory List Names added. To create a new Dependency Directory Name follow the steps in Table CP-3.

 

 

Table CP-3  Adding a Dependency Directory List Name

Step

Actions

1

From the main menu, select Windows – Perspective – OM Meister.

2

Select the Project Explorer tab located near the top of your screen.

3

Select and Right Mouse Click the Project which you want to assign a new Dependency Directory Name.

4

Select New – Dependency Directory to bring up the Dependency Directory Wizard.

5

Enter the Dependency Directory Name and select Finish

6

Your new Dependency Directory Name will be displayed under your Project Name on the left side of your screen.
