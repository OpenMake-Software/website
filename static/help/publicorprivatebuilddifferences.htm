Public or Private Build Differences

 

The Build Differences reports show what changed between the current build and the last build that was executed.  When the Workflow includes the use of an SCM Inspector Activity, such as the CVS Bill of Material Inspector, a Differences report will also be placed on the Knowledge Base server.   The SCM Inspector Activity will compare the data captured from the previous run of the SCM Inspector Activity and create a Differences report.  The Differences report will show what files have been added, changed, or deleted since the last run.  The files that are reported upon are ones located in the build directory only.   To view the report, use the Report Explorer from the OM Workflow perspective.  Your report will be available under Differences.

 

The data used to prepare the Differences report is stored in the build directory in files named as follows:

 

<Workflow>-<Machine>-<Job Date/Time>-bom.data and <Workflow>-<Machine>-<Job Data/Time>-.diff.data.

 

Note that these files should not be deleted from the Build Directory. The following are examples of how these file names might appear:

 

METALWORKS-My Computer-2006-03-01 12_00_01-bom.data

METALWORKS-My Computer-2006-03-01 12_00_01-diff.data
