Connecting to the PostgreSQL Database

 

In order to enable the PostgreSQL Database, it must be first installed on the Knowledgebase Server.  The Installation Zip File does not include the PostgreSQL Database.  The PostgreSQL Database can be installed on the Knowledgebase Server before or after the Knowledgebase Server components have been installed.  Once both the Knowledgebase Server and the PostgreSQL Database have been installed, you will need to change the configuration of the Knowledgebase Server so that it knows to communicate with the PostgreSQL Database.  To change the configuration of the Knowledgebase Server, refer to the  "Changing the Configuration" section of this Setup and Installation Guide.
