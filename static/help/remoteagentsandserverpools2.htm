Remote Agents and Server Pools

 

Remote Agents and Server Pools, available as an upgrade, are implemented to allow teams to execute Build and Workflow Activities from a pool of remote machines.  Remote Agents and Server Pools support the ability to execute Activities in a Workflow across multiple operating systems and machines.  Server Pools can include both physical and virtual machine configurations.

 

Every Knowledgebase Server can act as Remote Agent, however you must purchase a Remote Agent license key to activate the Remote Agent features of your Knowledgebase Server.  In addition, you can purchase as many Remote Agents as needed for your unique Build requirements.
