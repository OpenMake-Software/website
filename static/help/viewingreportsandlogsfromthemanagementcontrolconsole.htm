Viewing Reports and Logs from the Management Control Console

 

Instead of accessing reports via the Eclipse Rich Client Platform, you can also view Reports from the Management Control Console. 

 

OpenMake Meister specific Reports can be viewed from the top Menu listed as:

     Build Service Configuration Summary

o  Build Service Options – Shows Compile and Link options by Build Service.

o  Dependency Directories – Shows Dependency Directories by Project.

     Administration

o  Defined Users with Privileges – Shows Users and their Privileges.

o  Defined Groups – Shows Users and their Groups.

 

Workflow Logs and Reports can be viewed and accessed from the Workflow Results Summary Panel.  Use the “Search Workflow Results” to find your particular Workflow.  Each Workflow is displayed based on Run Number, Date/Time, Status and Machine.  Select the hyperlink of your desired Workflow to display the Workflow Results Summary and all corresponding Reports and Logs.
