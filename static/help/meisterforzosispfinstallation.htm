Meister for z/OS ISPF Installation

 

Installation of the z/OS components involve executing a REXX exec script that prompts the user for input regarding the installation datasets.  The installation dataset must be created prior to the execution of the install script.  Below are the requirements and steps for installing the z/OS components.

 

Software Requirements

 

•       z/OS 01.02.00 - FMID HBB7705

•       JES2 level z/OS 1.2

•       TSO level 2.6.0

•       ISPF level z/OS 01.02.00

•       VTAM level 6.1.2

 

Read/Write Requirements

No end user needs write access to any of the installation files.
