Apache Archiva Repository

 

Activity Plug-in

Description

Resolve Transitive Runtime Dependencies

Archiva can be used to resolve Transitive Runtime Dependencies during your compile and Link process.  The integration with Archiva is done at the Meister Build Method level. This means that in order to use this Plug-in, you must be using OpenMake Meister’s Build Services (not available in Mojo).   Please see “Using Archiva for Transitive Dependencies” in the “Creating Targets for Java IDEs and Development Tools” section of the Meister User Guide.  If you are not Using Meister, you will need to develop your own Maven or Ant script to call Archiva.
