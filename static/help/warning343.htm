WARNING 343:

              Could not find Scanner for <file extension>.

Description:

During scanning, the file extensions are mapped to a scanner via the scanner.ini file located in the <install dir>/perl/lib directory.  If you have source code that contains file extensions that are not mapped to a scanner in the scanner.ini file, you will receive this message.

Resolution:

To resolve this problem, update the scanner.ini file to map the extension to the correct scanner.
