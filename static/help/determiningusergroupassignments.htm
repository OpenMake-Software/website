Determining User Group Assignments

In order to quickly determine the Groups to which a particular User may already be assigned, open the Users Explorer from the OMAdmin perspective, and expand the node for the User. By expanding the User Node, you will see the Groups to which the User is currently assigned.  If you are an Administrator, you can use the User Editor to modify the assignment of a particular User to particular Groups.  You can add the User to a new Group or remove them from an existing Group by selecting and deselecting the checkboxes next to the Groups to which they are associated.  This feature is particularly useful in the case where a User was previously added to more than one Group, and now needs to be deleted from one or more of Groups due to a change in job duties.  Instead of searching through each Group for the User's name, you can simply go directly to the User and remove them from the Groups to which they no longer need access.
