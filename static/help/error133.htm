ERROR 133:

File <file name> is missing parameter for required option <build task option name>. Please add the option parameter to the target definition file.

 

Description:

An option was set by an Administrator at the parent build task level that requires a parameter; however the parameter was not defined as part of the Target Definition process.

 

Resolution:

Define the required parameter for the parent option of the Target being built or have an Administrator remove the requirement from the parent build task.
