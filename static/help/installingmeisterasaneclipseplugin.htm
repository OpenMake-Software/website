Installing Meister as an Eclipse Plug-in

Meister can be run inside of Eclipse Based IDEs providing a seamless interface into the IDE.  The following components must be installed and configured on your workstation prior to the installation of the Eclipse Plug-In:

•     IBM Rational Application Developer 7.0

•     Eclipse 3.1, 3.2

 

You can install Meister as an Eclipse Plug-in by using the Eclipse Software Update Manager.

 

Table SI-12  Installing via Software Update Manager
