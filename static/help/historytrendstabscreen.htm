History/Trends Tab Screen

 

The History/Trends Tab screen displays two sections, Workflow History and Data sections. The Workflow History section will display pass runs of the Workflow with the Result, Success or Failed, Start Date and Time and the Duration.  Double clicking on a particular row in the table will open the Summary and Detail reports for that Workflow run.

 

 The Data section displays the Last Execute Date and Last Execute Number which are the date the last time the Workflow was run and the last Build Number.  Also, hyperlinks to the prior Workflow logs and reports are available by clicking on the

View the “Workflow logs associated with prior executions” and View the “Workflow reports associated with prior executions” fields.
