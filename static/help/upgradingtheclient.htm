Upgrading the Client

The GUI updates are similar to Eclipse since it is based on the Eclipse Rich Client Platform. Updates can be made directly over the internet using a simple process.  For incremental upgrades from the previous version see Table SI-13.  For upgrades from 7.1 to the current version see Table SI-14.

 

Table SI-13  Upgrading via the Software Update Manager

Step

Action

1

In the GUI go to Help > Software Updates > Find Updates…

2

Choose “Search for new features to install”

3

Select the “Next” button

4

Select the “New Remote Site” button

5

Enter “Openmake Software” for the name and https://www.openmakesoftware.com/downloads/update_site for the URL

6

Select “OK”

7

Make sure “Openmake Software - Meister” is checked and then select the “Finish” button

8

Expand the “Openmake Software” node and select Meister

9

Select the “Next” button

10

Select “I accept the terms in the license agreement”

11

Select the “Next” button

12

Select the “Finish” button

13

Select “Install All” button

14

Select “Yes” button to restart the GUI.

 

 

 

Table SI-14  Upgrading  the DDS via the Software Update Manager

Step

Action

1

In the GUI go to Help > Software Updates > Find Updates…

2

Choose “Search for new features to install”

3

Select the “Next” button

4

Select the “New Remote Site” button

5

Enter “Openmake Software” for the name and https://www.openmakesoftware.com/downloads/update_site for the URL

6

Select “OK”

7

Make sure “Openmake Software - Meister” is checked and then select the “Finish” button

8

Expand the “Openmake Software” node and select Meister

9

Select the “Next” button

10

Select “I accept the terms in the license agreement”

11

Select the “Next” button

12

Select the “Finish” button

13

Select “Install All” button

14

Select “No” button to restart the GUI.

15

Edit the <install>/client/buildserver/rbs.ini and change the JRE_DIR to point to JRE 1.5 or higher.

16

Edit the <install>/kbserver/kbs.ini and change the JRE_DIR to point to JRE 1.5 or higher.

17

Switch the Java Version to 1.5 or higher using the PATH environment variable or use the –vm <java executable> parameter when starting Meister.

18

Run the UI and finish the upgrade process.
