clean

Deletes all final and intermediate Targets.

 

Note:  OOMLOM requires the use of .def files (Windows) and .exp files (AIX).  OOMLOM will create these files if they do not exist and will copy them to the build directory.  OOMLOM Clean will not delete these files, because the file may have been copied there by a developer and may not have been created by the build.

None required
