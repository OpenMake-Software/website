Assigning Remote Agents to the Server Pool

Once you have created a Server Pool, you will want to assign at least one Remote Agent. If you have properly configured and licensed Remote Agents, they will automatically appear in the Remote Agent table on the “Administer” panel found under the “Server Pool” menu option on the OpenMake Management Control Console. If you do not see any Remote Agents, please make sure they have been licensed and properly configured. See the Setup and Installation guide for more information on configuring a machine as a Remote Agent.

 

 

 Note: You must use the Management Control Console to assign Remote Agents to Server Pools. This feature is not available in the Meister Eclipse RCP Client.
