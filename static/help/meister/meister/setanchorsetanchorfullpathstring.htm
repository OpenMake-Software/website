setAnchor(), setAnchor( $full_path_string )

This method sets the anchor to the value of the $path_string variable.  Any future calls to getRelative or isRelative will interpret what is relative according to this new value.

 

If called with no argument, the anchor will be reset to the current working directory according to Perl.

 

Warning: there is no check to see if $full_path_string is itself an absolute path.
