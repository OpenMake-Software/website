Login Process

 

Each time a new User logins into the Eclipse RCP Client or Web Management Control Console, they will be prompted to enter their user ID and password.  The first time a User enters their User ID and Password, it will automatically be saved to the Knowledgebase Server.  New  Users should use their computer “login” name for initially logging into the Eclipse RCP Client or Web Management Control Console.  If you are using Windows, your computer “login” name is the same as what you enter when you boot Windows. For Unix or Linux environments, your computer “login” name is the same as your Unix or Linux account name. 

 

For the first time login, you can use any password.  The password that you initially enter will be saved on the Knowledgebase Server and encrypted for future logins.  You will be required to enter your Password twice to validate you have entered it correctly.
