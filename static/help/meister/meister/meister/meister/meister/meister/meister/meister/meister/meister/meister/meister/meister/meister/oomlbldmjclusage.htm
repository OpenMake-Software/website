OOMLBLDM JCL Usage

 

OOMLBLDM generates a Build Control file that is based on Project and Search Path.  You can modify the way OOMLBLDM executes by passing OOMLBLDM specific options using JCL.  In addition, you can generate a Build Control file for specific Targets by passing the specific Target Name to OOMLBLDM. You can also generate a Build Control file for an entire Project by passing no Target Names to OOMLBLDM.
