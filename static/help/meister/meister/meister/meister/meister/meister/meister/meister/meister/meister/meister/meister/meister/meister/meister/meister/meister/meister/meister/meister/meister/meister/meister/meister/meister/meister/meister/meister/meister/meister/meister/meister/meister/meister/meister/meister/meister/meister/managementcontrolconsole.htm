Management Control Console

The Management Control Console is a web based portal available for non-developers.  You can execute Workflows, access the Binary Repository, configure Server Pools as well as view Reports, Trends and Logs using the Management Control Console feature.  The Console is useful for individuals who do not need to define or configure Workflows, but simply want to execute Workflows and view the results.  Unlike the Eclipse Rich Client Platform, the Management Control Console is a URL that can be accessed by all users without needing to install components locally to their machine.
