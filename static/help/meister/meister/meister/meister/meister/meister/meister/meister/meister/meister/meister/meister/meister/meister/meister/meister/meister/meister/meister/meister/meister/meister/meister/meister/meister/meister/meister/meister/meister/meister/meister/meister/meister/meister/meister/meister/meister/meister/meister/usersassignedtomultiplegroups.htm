Users Assigned to Multiple Groups

 

Users can be assigned to more than one Group.  If a User is assigned to more than one Group, the highest-level privileges of those Groups will be allowed for that User.  All Users are by default added to the User Group.  If a User is also added to Groups with additional access privileges to certain Editors, they will have that additional access to only those specific Editors.  The remaining access privileges will be based on the default Users Group access.
