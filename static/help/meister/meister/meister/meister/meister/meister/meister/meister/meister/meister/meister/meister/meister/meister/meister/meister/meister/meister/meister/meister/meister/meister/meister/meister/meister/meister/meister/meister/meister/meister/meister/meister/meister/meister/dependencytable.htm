Dependency Table

 

All high-level source code modules need to be added to the Target Definition.  High-level source code includes any source, libraries or resource files that cannot be derived through the scanning process.  All included or "copy" files can be derived from the high level source, and therefore do not need to be added to the Target definition.  In essence, the list of Dependencies should include all files that are required as Source Code for a particular Target, and that cannot be derived by scanning other files for the information. 

 

OOMLOM has scanners for the following programming languages: C, C++, COBOL, PROC, PROCOBOL, and PL1.
