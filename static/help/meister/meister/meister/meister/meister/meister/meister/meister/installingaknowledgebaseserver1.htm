Installing a Knowledgebase Server

 

If the Knowledgebase Server was not installed by default then a manual install can be performed.  A manual install would be performed if you need to have multiple Knowledgebase Servers other than a centralized Knowledgebase Server.  Multiple Knowledgebase Servers are used when personal, testing, or production servers are desired. 

 

To install a Knowledgebase Server manually, first install Meisterite as directed above then perform the following steps:

 

Table SI-2  Installing a Knowledgebase Server

Step

Action

1

Launch OpenMake Meister

2

Go to Window > Preferences

3

Expand the Openmake item and select Machine

4

Under “This machine runs as a …”,  check Knowledgebase Server

5

Click “Install Knowledgebase Server Files”

6

Perform the configuration of Knowledgebase Server as described in the Changing the Configuration section of this guide.
