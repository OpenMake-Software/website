Configuring Workflow Activities

 

Workflow Activities can be specifically configured for your unique environment. In most cases, you will need to pass unique parameters to the pre-defined Workflow Activities so they are relevant. You can configure Workflow Activities to pass parameters or scripts to the underlying programs which they call and execute. By selecting an activity, you will be displayed the parameters required for that specific Workflow Activity.    Table WM-11 walks you through the steps of configuring the “Update From CVS” pre defined Workflow Activity.

 

Note: An asterisk by an input line within the Command field denotes required information.

 

Table WM-11 Configuring an Activity

Steps

Actions

1

Click your newly created activity, Update from CVS $(JOB_NAME). The Activity Information and Command fields will appear to the right of the activity.

2

In the Command field, type in the root directory of your CVS repository on the CVS Root input line.

3

Type in the general CVS options on the CVS Options input line.

4

Type in the options that are specific to your Update from CVS $(JOB_NAME) activity on the Update Options input line.

5

Go to the File menu and click Save.
