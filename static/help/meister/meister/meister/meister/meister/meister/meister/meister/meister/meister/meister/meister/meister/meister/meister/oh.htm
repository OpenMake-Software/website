-oh

Sends Build Log Output to an HTML file stored on the Knowledgebase Server.  Must use the following to specify the log file name:

 

 –ld  <yyyy-mm-dd hour:minute:second>

 

Note: You must specify your date and time. The command line does not use the actual system date and time. This date and time will be used to create your file name.

 

-lj  <job name>

-lm <machine name>

-lo <owner name>

 

example:

 

OOMLOM -ob –ld "2001-01-01 01:01:01" –lj "build demo Development" –lm LAB01 –lo MTOM04

 

Also, if the Build should be logged as public pass:

-lp
