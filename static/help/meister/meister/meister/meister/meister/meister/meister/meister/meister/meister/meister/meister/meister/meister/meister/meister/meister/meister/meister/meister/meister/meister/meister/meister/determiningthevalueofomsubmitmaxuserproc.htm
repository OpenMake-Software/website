Determining the Value of OMSUBMIT_MAX_USER_PROC

 

The value of OMSUBMIT_MAX_USER_PROC  is the maximum number of simultaneous Workflow Activities that your machine can run in multi-threaded mode.  As a general rule, each processor can handle a maximum of four (4) Workflow Activities.  This is often dependent upon the type of Workflow Activities you are executing and the processing power of your machine.   It is recommend that you begin by setting the value to a lower number and experiment with your processing speeds until you begin seeing no further improvements.  Following is a recommend setting:

 

Machine Configuration

OMSUBMIT_MAX_USER_PROC

Maximum Recommend Value

Single Processor

4

Dual Core

8

Quad Core

16

4 Dual Core Processors

64
