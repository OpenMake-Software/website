Archiva Configuration

 

Archiva provides web services that are used by the Meister Ant War Build Method to find each archive’s dependency tree.  In order for the web services to return the dependency tree for an archive that archive must reside in an Archiva Repository and the Repository must be indexed to include the archives information.  The Archiva documentation for adding to and indexing a repository can be found at:  http://archiva.apache.org/docs/1.3/userguide/index.html and http://archiva.apache.org/docs/1.3/adminguide/index.html
