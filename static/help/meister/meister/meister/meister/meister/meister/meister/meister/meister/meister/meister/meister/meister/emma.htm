Emma

Activity Plug-in

Description

Emma Instrument Classes

This Workflow Activity executes code coverage test against Java classes.

Emma Report

This Workflow Activity generates an Emma Code Coverage Report.

Emma Runtime Instrumentation

This Workflow Activity executes Emma in Runtime mode versus offline mode.

 

The source code directory is passed in as a parameter

The compiled classes classpath are passed in as a parameter

•    should be comma separated

•    accepts both directories and jar files

 

Install Instructions:

1.   Download the latest version of Emma from sourceforge

      - http://emma.sourceforge.net/

 

2.   Select the downloads link from the menu on the top left

-     http://sourceforge.net/project/showfiles.php?group_id=108932

3.   Select the download link for the Emma Release package

4.   Select the emma lib zip file and download it

5.   Unzip the file to a local directory on your hard drive

6.   Once unzipped copy the jar files in the unzipped directory  to <MEISTER_INSTALL_PATH>/client/tools/emma/lib
