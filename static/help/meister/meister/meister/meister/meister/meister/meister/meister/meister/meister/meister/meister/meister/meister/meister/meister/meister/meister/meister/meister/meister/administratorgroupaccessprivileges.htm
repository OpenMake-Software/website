Administrator Group Access Privileges

 

Only the default Administrator Group can update the Views associated with Administrator privileges.  Administrator functions include the following:

 

•  Build Service Explorer - Allows Administrators to define and customize Build Services.

•  Project Explorer - Defines Projects and Project Dependency Directories

•  Manage Groups - Defines Groups, Group Privileges and Group Users.

•  Manage Users - Tracks Users and their associated Groups.

 

*Note:

If you have enabled LDAP for your Knowledge Base server, each of these objects (Build Services, Projects and Dependency Directories, and Workflows) will have a corresponding entry in the LDAP node for Read, Write, Add, Execute, and Delete privileges.

 

Only the Administrator Group can create Projects and manage Groups and Users.  These functions cannot be assigned to another Group.
