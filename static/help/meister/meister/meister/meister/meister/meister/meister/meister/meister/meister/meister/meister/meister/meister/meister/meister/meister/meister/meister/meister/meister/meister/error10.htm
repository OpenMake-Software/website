ERROR 10:

          Could not open depend.xml

Description:

The -g option was passed to om. The -g option gathers dependencies for Impact Analysis and saves it to a file called depend.xml.  This message indicates that the depend.xml file could not be written to.

 

Resolution:

Make sure that if there is an existing depend.xml file, it can be written to.  It may have been changed to read only. Also, check to make sure that there is space available to write the depend.xml file.
