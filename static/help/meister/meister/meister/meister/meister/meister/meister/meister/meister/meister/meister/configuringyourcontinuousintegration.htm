Configuring your Continuous Integration

 

Once you have installed your Remote Agents as needed, or you have chosen to use your local machine as your continuous integration server, you can configure how your Continuous Integration process will execute for each Workflow.  By using the Continuous Integration Tab from the Workflow Editor (available only in the Eclipse Rich Client Platform), you can associate the Source Code Management tool which you will be using to trigger your Continuous Integration process, assign a Continuous Integration Server and determine the frequency in which your Continuous Integration process will execute.
