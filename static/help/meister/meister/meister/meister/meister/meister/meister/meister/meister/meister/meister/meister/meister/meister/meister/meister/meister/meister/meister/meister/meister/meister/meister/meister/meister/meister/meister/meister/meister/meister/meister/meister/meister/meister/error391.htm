ERROR 391:

Could not determine final dependency extension for Target <Final Target Name>

Description:

The Final Target extension in Build Service does not match the Final Target in the TGT. This occurs when the Build Service has been modified and that the TGT has not been updated.

Resolution:

Verify that the Build Service Rules specify the correct Final Target extension.  From the Target Explorer, edit the Target and define a matching Final Target that has an extension that matches the definition in the Build Service.
