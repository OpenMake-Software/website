GenerateFootPrint($FootPrint,$TargetFile,$FPSource,$FPObject,$CompilerFound,$FPCompilerArguments )

 

This function generates the footprinting source file for the specified target file. This source file is compiled and linked into the target.

 

USAGE:

 

 GenerateFootPrint( $FootPrint,$TargetFile, $FPSource,$FPObject,

    $CompilerFound,$FPCompilerArguments );
