Upgrading the Knowledgebase Server

The Eclipse update process used to upgrade the client will also upgrade the server if it is installed on the same machine. To upgrade the Knowledgebase server, go to the machine that has the Knowledgebase Server installed and follow the upgrade instructions for the client. This will also update the Eclipse plug-ins for the Knowledgebase Server, which in turn triggers the actual server updates when the GUI is restarted.
