OpenMake Meister

 

Activity Plug-in

Description

Execute Meister Incremental Build

This Workflow Activity is used to execute om in Incremental mode to run the build, building only those components that are out of date.  The Workflow Activity contains fields for options, targets, and variable settings.

Execute Meister Incremental Build with Audit Report

 

This Workflow Activity is used to execute om in Incremental mode, building only those components that are out of date, while generating a Build Audit Report.  The Workflow Activity contains fields for the Audit Report name, Version Program, options, targets, and variable settings.

 

Meister Build Control File

 

This Workflow Activity uses OpenMake Meister to generate the Build Control File.  The Workflow Activity contains fields for Project, Dependency Directories, and Flags.
