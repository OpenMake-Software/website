Uninstalling Meister

To uninstall, perform the following steps:

 

Table SI-15  Uninstalling

Step

Action

1

Stop the Knowledgebase Server and Build Servers if they are running

2

Run the uninstall program found in the <install root>/openmake-uninstall directory

 

The uninstaller will delete in the files in the <install root> and remove the property files from the user’s home directory.
