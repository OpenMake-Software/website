Resetting Passwords

The User ID and Password information is stored on the Knowledge Base Server in a file called user_validation, stored in the following location:

 

<Installation directory>/kbserver/tomcat/webapps/openmake.ear/openmake.war/

 

In order to reset a password, the file can be edited to delete the line containing the User ID and Password.   By deleting both the User ID and Password from the file, the User will be allowed to re-enter their new User ID with a new Password when they login to the Eclipse RCP or Web Management Control Console.
