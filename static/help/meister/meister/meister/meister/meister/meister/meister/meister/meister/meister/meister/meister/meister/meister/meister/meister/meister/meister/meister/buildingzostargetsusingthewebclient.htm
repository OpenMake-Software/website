Building z/OS Targets using the Web Client

 

Builds can be executed directly from the Web Client from the “Execute Workflow” menu option. 

 

Note:  Builds for the mainframe are not supported via the “Setup Build” menu option from the Web Client.  This feature is replaced by the ISPF native panels.
