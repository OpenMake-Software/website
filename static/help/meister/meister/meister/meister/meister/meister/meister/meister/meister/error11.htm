ERROR 11:

          Could not change directory to <directory name>

Description:

The -c option was passed to om. The -c option changes the directory where om will execute.  The -c option requires a parameter that is the name of the directory where bldmake will execute.  This error displays when the directory is not available or cannot be written to.

 

Resolution:

Make sure the directory exists or that it is not a protected, read only directory.  You should also make sure that there is space available in the directory.
