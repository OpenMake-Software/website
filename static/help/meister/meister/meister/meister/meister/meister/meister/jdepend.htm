JDepend

JDepend traverses Java class file directories and generates design quality metrics for each Java package. JDepend allows you to automatically measure the quality of a design in terms of its extensibility, reusability, and maintainability to manage package dependencies effectively.  This Workflow Activity will add calls to JDepend to be added as a step in your Workflow.

 

The compiled classes classpath is passed in as a parameter

•    should be comma separated

•    only directories are accepted

 

Install Instructions:

1.   Download the latest version of JDepend from http://clarkware.com/software/JDepend.html

2.   Select the JDepend link under the Downloading JDepend section from the link above

3.   Unzip the file to a local directory on your hard drive

4.   Once unzipped copy the jar files from the lib directory  to <MEISTER_INSTALL_PATH>/client/tools/jdepend/lib
