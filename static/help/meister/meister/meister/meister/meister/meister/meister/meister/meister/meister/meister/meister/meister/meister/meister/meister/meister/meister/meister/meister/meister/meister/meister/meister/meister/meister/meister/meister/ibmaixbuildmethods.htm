IBM AIX Build Methods

 

Build Method Name

Description

AIX ar Archiver.sc

Executes ar to create an archive file ( .a )

AIX C and Cpp Compiler.sc

Executes the IBM Compiler xlC, xlc, xlC_r and xlc_r to create an .o  from a C/C++ source

AIX C and Cpp Linker.sc

Executes the IBM Linker or xlC, xlc, xlC_r and xlc_r to create an executable or shared library from the objs and libraries

IBM mqsicreatebar Compiler.sc

 

Executes the WebSphere WBIMB mqsicreatebar

Visual Age C Project Compiler.sc

 

Execute the IBM Visual Age C compiler against a Visual Age Project file to create the objs
