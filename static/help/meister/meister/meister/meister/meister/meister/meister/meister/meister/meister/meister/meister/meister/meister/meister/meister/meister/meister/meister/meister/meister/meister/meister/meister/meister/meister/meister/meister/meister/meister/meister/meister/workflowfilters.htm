Workflow Filters

The Workflow filter allows you to limit the number of Workflows that you see in your available Workflows.  This also limits which Workflows will be shown in the Workflow Monitor.  Your Administrator can determine which Workflow that you have access to; however, you can further limit the Workflow that you wish to see by using the filter. 

 

The Workflow filter is available in both the Eclipse RCP and the Web Management Console.  In the Eclipse RCP you will see a “funnel” icon near the Workflow Explorer menu tab.  Click on the “funnel” icon to display a dialog box that will allow you to select which Workflows to Hide.  By default, all Workflows are “shown.”  Move all Workflows that you wish to “hide” to the Hide column. If they show in the Hide column, they will not be displayed in the Workflow Explorer and you will not see them in the Workflow Monitor.  The Workflow filter available via the Web Management Console works in the same way. To access your list of Workflows, go to the “Filter Setup” menu option under, “MyWorkflows.”
