OpenMake Meister for z/OS

Meister for z/OS provides all the functionality of OpenMake Meister on the z/OS platform.  Meister for z/OS serves as standard compile processors for languages supported on the z/OS platform.  Managing libraries, compile flags and Load Libraries are simplified using Meister for z/OS. No private compile JCL is required.
