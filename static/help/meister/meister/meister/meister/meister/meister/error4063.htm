ERROR 406:

          Could not find Build Service

 

Description:

The Build Service associated with the Target Name is no longer valid.

Resolution:

The Build Service may have been deleted or renamed since the last this Target was compiled.  Verify that the Build Service is still valid by viewing the available Build Services through the Build Service Explorer.  If the Build Service is no longer available, contact your administrator to determine the new Build Service that should be used by your Target.
