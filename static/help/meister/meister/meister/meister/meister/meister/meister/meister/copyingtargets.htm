Copying Targets

Although Targets are associated with Projects, it is possible to copy Targets and assign them to other existing Projects.  In such cases, updates and customizations will normally need to happen to actually use the copied Target for another Project build.

 

To copy a Target, right mouse click on the Target from the Target Explorer and select Duplicate.  You will be brought to a new Overview Target Screen. You will need to change the name of the Target as well as the name of the Target file (.tgt).
