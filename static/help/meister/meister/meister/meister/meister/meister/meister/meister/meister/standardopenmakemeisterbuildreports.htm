Standard OpenMake Meister Build Reports

 

You are provided with several standard reports that display Build, Deploy and Workflow results and metrics, including:

         

•         Public Workflow Logs

•         Private Workflow Logs

•         Workflow Summary

•         Workflow Graphs

•         Workflow Activity Summary (Requires PostgreSQL Database)

•         Public Build Activity

•         Private Build Activity

•         Public Bill of Material Reports

•         Private Bill of Material Reports

•         Public Build Difference Reports (requires version control repository)

•         Private Build Difference Reports (requires version control repository)

•         User Groups

•         User Privileges

•         Insight Public Build Audits

•         Insight Public Implosion Impact Analysis (full Meister license required)

•         Insight Public Explosion Impact Analysis (full Meister license required)

•         Insight Private Build Audits (full Meister license required)

•         Insight Private Analysis Implosion Impact (full Meister license required)

•         Insight Private Analysis Explosion Impact (full Meister license required)

•         Build Service Options (full Meister license required)

•         Dependency Directory Usage (full Meister license required)
