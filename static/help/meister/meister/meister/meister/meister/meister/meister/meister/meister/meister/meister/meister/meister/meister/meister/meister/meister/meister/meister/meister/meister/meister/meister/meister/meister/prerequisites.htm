Prerequisites

The following configuration has been tested with:

 

•     Eclipse 3.4.2

•     Eclipse C/C++ Development Tools version 5.0.2.200902130801

•     Meister 7.3

•     Any gcc or Microsoft Visual Studio tool chain supported by CDT, including make programs such as GNU make or Microsoft’s nmake.

•     A Meister project and dependency directory to locate the Eclipse workspace using the WORKSPACE_ROOT and TARGET_DEFINITION parameters.

•     Meister Targets created for the build. For more information, see Defining C/C++ Targets for Eclipse CDT under Creating and Managing Targets.

 

Note:  For simplicity, only make will be referred to in the documentation, with the understanding that nmake can be used as an alternative.
