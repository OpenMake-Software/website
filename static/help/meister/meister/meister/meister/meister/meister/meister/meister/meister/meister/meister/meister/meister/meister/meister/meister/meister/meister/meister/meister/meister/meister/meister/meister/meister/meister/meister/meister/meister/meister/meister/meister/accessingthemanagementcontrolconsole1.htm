Accessing the Management Control Console

 

The Management Control Console is a web based portal that allows you to execute Workflows, change the Status of a Builds, define and configure Server Pools and view Workflow History and Trends.  The Management Control Console is helpful for individuals who require a self-service portal to executing and viewing Workflow and build results but cannot access the Meister Eclipse Rich Client Platform (RCP).  The Management Control Console is a URL that can be accessed by all users.

 

You can access the Management Control Console from your Windows “Start” menu or use the following URL:

 

http://<KB Server Name>:<KB Server Port>/openmake/index.html

 

For example:

 

http://winkb:58080/openmake/index.html

 

 

It is not necessary to install any software locally to access the Management Control Console.
