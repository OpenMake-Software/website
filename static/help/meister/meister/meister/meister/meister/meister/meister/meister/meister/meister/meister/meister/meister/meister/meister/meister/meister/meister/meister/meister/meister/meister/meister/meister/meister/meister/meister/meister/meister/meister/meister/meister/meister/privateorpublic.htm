Private or Public

 

A Workflow is set up as either a Public Build or a Private Build (MyWorklfow).  A Workflow is set up as Public in order to allow other Users to execute the Workflow.  A Workflow is set up as Private in order to allow only the creator to execute the Workflow.  Mark "Public" or "Private" on the Workflow Detail screen to indicate if you want to share your Workflow with others, or keep the Workflow private, respectively.
