Overview Attributes

 

Overview attributes are used to identify the Build Service by name and file extension of the binary you are using the Build Service to create.

 

Build Service Name

 

The name should use the binaries file extension to assist in identifying what the Build Service creates.  For example: Java Jar.

 

Operating System

 

Select from the available Operating Systems what OS will be used by your Build Service.

 

Final Target Extension

 

Once defined in the Detail panel, the final target extension will be displayed.
