Using the Build Method Editor

 

The Build Service Editor comes equipped with a Build Method Editor that allows you to edit Build Methods. The Build Method Editor comes with syntax highlighting of the Build Methods that you are editing, and access to documentation for each Perl module available on the local machine, including Build Method specific modules, as well as general modules available with your local Perl installation.

To access the Build Method Editor, highlight the Build Rule for which you want to edit a Build Method. Click the Build Method to Execute link to open a Build Method Editor for the Build Method. A new editor appears with the Build Method you want to edit. The syntax of the Build Method is highlighted to make Build Method editing easier.

If you need to reference documentation for a particular Perl Module, click on the Perl Doc tab in the Build Method Editor. In the Perl Doc tab, a list of all available Perl modules appears in a drop down field. Select the Perl module you want more documentation about from the drop down.
