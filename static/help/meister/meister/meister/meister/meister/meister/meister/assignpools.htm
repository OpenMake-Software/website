Assign Pools

 

If you have purchased and configured Server Pools and Remote Agents you can define your entire Workflow to execute on an available Server Pool, or select a specific Server

Pool for a single or set of Workflow Activities.  By clicking on the “Assign Pool” button, found under the Activities Tab, you will see the available Server Pools and “MyComputer”, your local machine.  You can assign these available Server Pools to your Workflow, or Workflow Activities.   Without Remote Agents and Server Pools, you will only be able to define Workflow Activities to execute on your local machine, My Computer.  When the Workflow is executed, it will execute all Workflow Activities on the Server Pools assigned.
