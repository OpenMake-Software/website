Creating Targets for the Microsoft Visual Studio and .Net IDEs

 

There are two ways to create Visual Studio and .Net Targets. You can use the Microsoft Add-In to automatically generate the Targets or you can manually create the Targets. It is recommended to use the Microsoft Add-in to create the Targets as it provides a more automatic and accurate process.   The Microsoft Add-in will automatically generate Target files based on the Microsoft project file and prevent you from having to do this step manually
