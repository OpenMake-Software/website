Monitor Tab Screen

 

The Monitor Tab screen displays the real-time execution of the Workflow selected, as well as a sub-tab that show a “Global” Workflow monitor which shows all Workflows currently running that you have access to and have not filtered. In addition, you can view a Summary of all you Workflows using the “Workflow Summary” sub-tab, or view “Workflow Logs” using the Workflow Log explorer.   Double clicking on any of the rows will bring up the detail HTML log output.

 

The Workflow Monitor displays Workflow activities as they are being executed and displays the results of each individual Activity after it has been run.   Use the Workflow Monitor to show the Workflow progressing from one Workflow Activity to the next and the status of each Activity.  To execute and view the status of your Workflow, follow the steps in Table WM-9.

 

Table WM-9 Executing and Monitoring a Workflow

Steps

Actions

Eclipse RCP

 

1

Go to the OM Workflow Explorer Tab.

2

Right click on the Workflow from the Workflow Explorer and select Execute.

3

Monitor the execution of your Workflow on the Workflow Monitor screen, or from the OM Dashboard Perspective.

Management Control Console

 

1

Bring up the web based Management Control Console.  Select MyWorkflows > Latest Runs.  Highlight the Workflow to execute and click on the “Execute Selected” Button.

2

View the results by going to MyWorkflows > Workflow Monitor.

 

 

Each Activity displayed in the Workflow Monitor is color-coded based on its status.

 

Table Workflow Monitor Status

Color

Status

Red

Workflow activity failed

Green

Workflow activity executed successfully

Orange

Dependent activity in error.

Gray

Activity pending execution

Yellow

Activity in progress

 

 

You can also use the Meister client’s “OM Dashboard” perspective to monitor and view summaries of all Workflows for which you have access and have not filtered or view the monitor using the web based OpenMake Management Control Console found under “MyWorkflows”, “Workflow Monitor.”
