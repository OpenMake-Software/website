Workflow Activity Application Life Cycle Plug-in Reports

         

A Workflow Activity Life Cycle plug-in calls an external tool to perform a specific action, for example, the calling of a static code analysis tool, testing tool, email or deployment action.  All Workflow Activity plug-in Reports are generated and presented as provided by the external tool vendor.  Workflow Activity plug-in Reports can only be viewed from the Workflow Results Summary panel available in both the Eclipse Rich Client Platform and the Management Control Console.
