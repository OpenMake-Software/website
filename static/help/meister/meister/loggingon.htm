Logging On

In order to execute builds on the mainframe and report the build results, including impact analysis, back to the Knowledgebase Server, you must use an ID that has access to the Knowledgebase Server.  In some cases, your Mainframe User ID may be different from your LAN User ID.  If this is the case, be sure to enter your LAN user ID at the bottom of the  main ISPF panel as follows:

 

LAN USER ID

   === -> JDOE143

 

If your Mainframe and LAN ID are the same, this is not a required field.
