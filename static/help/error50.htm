ERROR 50:

          Could not find .dsp in Dependency Directory

 

Description:

          Bldmake could not locate the MSVC++ Project file in the Dependency Directory.

 

Resolution:

Insure that the Dependency Directory includes the directory containing the .dsp file. The .dsp file may be located in the local directory which is represented by a “.” in the Dependency Directory, or the .dsp listed as a dependency in the Target file maybe misspelled.
