Creating Groups

In order to create a Group, you must be an Administrator.  To create a Group, go to the main menu and select  Window - Open Perspective-OM Admin. The OM Admin perspective contains both the User Explorer and the Group Explorer.  The instructions in Table UG-1 will get you started.

 

Table GU-1. Creating a Group

Steps

Actions

1

Select the Groups Explorer tab, and retrieve the Administrative Data, if necessary. A list of Groups will appear in the Groups Explorer.

2

Right click the name of a Group, and select New from the context menu that appears. You will be prompted to enter the name of the Group.  Once you have entered the Group name, an Editor will be opened.

3

The Editor contains several options tabs. The Overview tab allows you to select which Users you want to belong to the Group. The Build Services Privileges, Dependency Directory Privileges and Public Workflow Privileges tabs allow you to manage privileges for individual features.
