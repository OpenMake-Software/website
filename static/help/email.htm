Email

 

Use this Workflow Activity to send e-mails when Workflows are being executed and to specify whether Workflows succeed or fail.  For this Workflow Activity to work properly, the Knowledge Base server needs to be properly configured to interact with a SMTP server. Please see the Mojo Install Guide for instructions for configuring your SMTP server.  All the e-mail fields have their standard function, except for the “Recipients” field, which requires entries to be comma delimited.  The message can be setup to be sent depending upon the success or failure of other activities under the Configuration tab for the Send E-mail Workflow Activity.  See Activity Dependencies and Sorting below for further information.
