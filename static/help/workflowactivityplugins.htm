Workflow Activity Plug-ins

Meister provides an ecosystem of Plug-in activities to support a complete life cycle, from development to release.  Pre-defined Plug-ins are provided for popular versioning tools, testing tools, static analysis tools and development tools.  Workflows automate repetitive processes by reusing Plug-ins, thus improving productivity. Developers can use Plug-ins to streamline their build to release Workflow without having to re-invent the wheel. Plug-ins provide out-of-the-box integration of the check-in, check-out, version labeling, version reporting, difference reporting, testing, static analysis and email. Plug-ins are flexible and can be customized by the user to fit any life cycle requirement.
