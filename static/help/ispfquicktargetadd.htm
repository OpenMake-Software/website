ISPF Quick Target Add

 

In some cases, particularly with COBOL modules, a Final Target may contain only one dependency.   In this case, the Quick Target Add (option 2) allows you to quickly create one Target for each source module in a data set.  When using this option, all Targets will be assigned the same Project, Build Type and Target Extension. 

 

After selecting the Project, Operating System, Build Type and entering the Target Extension and Source Code PDS, press your ENTER key.  This will create a Target file for each source module found in the Source Code PDS.  The TGT files will be saved in a PDS under the type TGT based on the High Level Qualifier.

 

The new Targets will be displayed under the Members list.
