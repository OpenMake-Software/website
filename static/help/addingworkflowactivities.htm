Adding Workflow Activities

 

Workflow Activities are the steps of your Workflow.  You can use pre-defined Workflow Activities or define your own unique Workflow Activities.  Several pre-defined Workflow Activities are available “out of the box” and can be displayed using the “Add Activity” Button.  The Workflow Activities are classified based on Tool name or by function, such as “Test Tools.”

 

 If you wish to have your Workflow Activities execute on a location other than “MyComputer” you should first assign a new Server Pool to your Workflow.  You can then assign Workflow Activities to the Server Pool.

 

Table WM-10 shows how to add a Workflow Activity using the “Update from CVS”  pre-defined Workflow Activity. 

 

Table WF-10 Adding Activities

Steps

Actions

1

Select the Activities tab.

2

Highlight "My Computer" from the Workflow Hierarchy field, which is located within the Workflow Editor screen.

3

Click the Add Activity button. A window appears with two fields: one for plug-ins (Activity Plug-in) and the other for activities (Activity Type).

4

Select the CVS plug-in from the Activity Plug-in field and then the Update from CVS activity from the Activity Type field.

5

Click OK. Your new activity name, Update from CVS $(JOB_NAME), will appear under the machine that you chose.
