Launching the Install

 

An install wizard will prompt you for the necessary information to complete the install.  This information can be changed at a later date by using Window – Preferences- Openmake -  Machine.

 

 

Table SI-1  Installing

Step

Action

1

Unzip the dds.zip file to a directory including all subdirectories.

2

Launch dds.exe

3

Install Wizard will display if OPENMAKE_SERVER is not set.  (If you need to change the install options once the install completes, use the Preference Page available from the Main Menu Windows – Preferences - Openmake.)

4

Click “I accept the terms of the License Agreement” on the License Agreement message box and hit Ok

5

Click Yes to the New Openmake Software Install message box to complete the install process

6

A Progress Information message box will display the extraction and configuration of Meister.  After the Progress Information message box disappears the software install is complete.

 

 

*Note: A local Knowledgebase Server will not be installed if the OPENMAKE_SERVER environment variable is set.  The Meister will utilize the Knowledgebase Server defined by the OPENMAKE_SERVER. The Command Line Client will be installed automatically.
