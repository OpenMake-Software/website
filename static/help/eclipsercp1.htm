Eclipse RCP

 

The graphical user interface is based on the Eclipse Rich Client Platform.  It is presented to the user in the form of Perspectives.  It is divided into 5 separate Perspectives:           

•     OM Admin – Management of Users and Groups

o  User Explorer and Editor

o  Group Explorer and Editor

•     OM Workflows – Management of Workflows and Schedules

o  Workflow Explorer and Editor

o  Workflow Log Explorer and Viewer

o  Deploy Services Explorer and Editor

o  Workflow Service Explorer and Editor

•     OM Dashboard - Provides an overview of all Workflows and the Workflow Monitor.

•     OM Reports – Presents Bill of Material, Metrics and Build Difference Reports.

o  Reports Explorer and Viewer
