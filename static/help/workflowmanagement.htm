Workflow Management

 

Workflow management releases the potential of enterprise teamwork through the coordination of multiple Application Lifecycle related steps from build to release. Workflows organize and automate your pre build, build and post build activities as a single process.  Workflows can be triggered, scheduled or executed on-demand.  Workflows can be executed across multiple Remote Agents and Server Pools for improved workload processing. As Workflows are executed, result logs are gathered in a central location for reporting.  In addition, you can monitor your Workflow progress in real time using the Workflow Monitor, and filter the Workflows  you see, using the Web Management Console.
