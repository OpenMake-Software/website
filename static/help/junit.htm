JUnit

JUnit is a simple framework to write repeatable tests. It is an instance of the xUnit architecture for unit testing frameworks.  This Workflow Activity will allow you to add JUnit to your Workflow.

 

The source code directory is passed in as a parameter

The compiled classes classpath is passed in as a parameter

•    should be colon or semicolon separated

•    jars and/or directories are accepted

The source to include is passed in as a parameter

The source to exclude is passed in as an optional parameter
