Linux and UNIX Installation Notes

 

For the following sets of installation instructions, note the following differences:

 

•     The name of the installer executable is “dds” without the “.exe” ending. References to launching dds.exe should be dds on Linux and UNIX.

•     After extracting the client into the desired installation directory, you may need to set the execution bit on the DDS executable using the chmod command as in “chmod 777 dds”. Note: verify the proper permissions and command to use with your company’s security department so you are in compliance with your company’s security policies.

•     A valid X-Windows display must be available prior to launching the installation, and the DISPLAY environment variable set accordingly.
