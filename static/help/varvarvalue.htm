<var>=<Var value>

Tells OOMLOM to pass variable information to the Command scripts.  If customized scripts are written, variables can be defined and the value can be set when OOMLOM executes.  OOMLOM will then pass the value of the variable to the Scripts.  The following pre-defined Variables can be passed to the scripts via Workflows or the Command Line when executing OOMLOM:

CFG = Release

CFG = Debug

Determines if Release or Debug flags are to be used.
