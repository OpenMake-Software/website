Installing a Remote Agent and Continuous Integration Server

 

The Remote Agent enables Workflows to be executed remotely and enable the Continuous Integration Server functionality.  You must purchase a license in order to use Remote Agents.  The Remote Agent is bundled in the DDS zip file for convenience.

 

To install a Remote Agent, first install Meister as directed above then perform the following:

 

Table SI-3  Installing a Remote Agent

Step

Action

1

Launch Meister

2

Go to Window > Preferences

3

Expand the Openmake item and select Machine

4

Under “This machine runs as a …”, check Remote Agent

5

Click “Install Remote Agent Files”

6

Perform the configuration of Remote Agent as described in the ‘Changing the Configuration’ section of this guide.

*Note:  A license is required to use any installed Remote Agents.
