Serena

Activity Plug-in

Description

Update from PVCS

This Workflow Activity takes updates from the PVCS file repository and applies them to your workspace, usually at the start of a Workflow before other activities are conducted. The Workflow Activity contains fields for options that should be filled in with the desired arguments.

 

Execute Meister Incremental Build with PVCS Audit

 

This Workflow Activity is used to execute a Meister Build, rebuilding only those components that are out of date with Build Audit Reporting exposing all items which are not managed in PVCS.  When items are found in PVCS  the version information of the source files from the PVCS repository is reported.    The Workflow Activity contains fields for the Audit Report name, options, targets, and variable settings. Note: You must execute a Meister Build Control File Workflow Activity before this step.

 

The PVCS_VER_LABEL environment variable is used to determine what PVCS version label to check against for the audit report. Set PVCS_VER_LABEL equal HEAD to report against the head revisions.

Execute Meister Clean All Build with PVCS Audit

 

This Workflow Activity is used to execute a Meister Build, rebuilding all components with Build Audit Reporting exposing all items which are not managed in PVCS.  When items are found in PVCS the version information of the source files from the PVCS repository is reported.  The Workflow Activity contains fields for the Audit Report name, options, targets, and variable settings.

 

The PVCS_VER_LABEL environment variable is used to determine what PVCS version label to check against for the audit report. Set PVCS_VER_LABEL equal HEAD to report against the head revisions.

 

Note: You must execute a Meister Build Control File Workflow Activity before this step.
