-rr

Retrieve Rules.  The Retrieve Rules file provides the Retrieve Program additional rules information for checking out files.  For example, it may be required to retrieve a particular version of a file.  The Retrieve Rules file should contain this information. The file specified by this parameter is passed to the Retrieve Program (-rp).  OOMLOM locates the rules file by searching the Search Path.

OOMLOM –rp <script name> –rc <configuration file name> -rr <rules file>

example:

OOMLOM –rp getfiles.bat –rc configfile.txt –rr rulefile.txt
