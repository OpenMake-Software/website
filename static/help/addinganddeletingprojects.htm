Adding and Deleting Projects

Managing Projects is done through the OM Meister Perspective.  From the main menu, select Windows – Perspective – OM Meister.  This will bring you to the OM Meister perspective where the Project Explorer tab is available.  Table CP-1 will take you through the steps of creating a Project

 

Table CP-1  Adding a Project

Step

Actions

1

From the main menu, select Windows – Perspective – OM Meister.

2

Select the Project Explorer tab located near the top of your screen.

3

From the main menu – select File – New - Project

4

Enter the Project Name and select Next

5

Enter the Dependency Directory Name and select Finish

6

Your Project will be displayed in the Project tree on the left side of your screen in alphabetical order.

 

 

Table CP-2  Deleting a Project

Step

Actions

1

From the main menu, select Windows – Perspective – OM Meister.

2

Select the Project Explorer tab located near the top of your screen.

3

Select and Right Mouse Click  the Project you wish to Delete

4

Select Delete.  The Project and all associated Dependency Directories will be deleted from the Project tree on the left side of your screen.
