Viewing Reports and Logs using the OM Workflow Perspective

 

Using the OM Workflow Perspective, right mouse click on your Workflow and select Edit/View.  This will display the Workflow History with all runs of this particular Workflow listed by Run Number, Start Date/Time, Duration and Machine.  Select the hyperlink to the desired Workflow to view the Workflow Results Summary.  All Reports and Logs can be accessed via the Workflow Results Summary.  If your Workflow executed a Workflow Activity that used an application life cycle plug-in, the associated Reports and Logs for the plug-in will be displayed in a menu format at the top of the Workflow Results Summary.
