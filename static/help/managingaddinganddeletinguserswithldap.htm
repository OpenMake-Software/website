Managing, Adding, and Deleting Users with LDAP

 

If you have LDAP enabled, all User management functions will take place within your LDAP administration program. Due to the structure of privileges on the Knowledge Base server, much of the work of managing Users is automatically handled based on default User Group.  If you want to change a User's privileges, all you need to do is assign that User to the appropriate group.

 

You can get a better idea of how privileges work on the Knowledge Base Server by reviewing the Groups section of this guide. Each set of privileges detailed below exists as an entry to the node on your LDAP server.
