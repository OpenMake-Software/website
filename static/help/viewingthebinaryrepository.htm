Viewing the Binary Repository

 

The binary repository can be viewed from the Web Management Control Console.  From the “Previous Runs” Menu option found under the “MyWorkflows” option, you can select a Workflow and view the collection of binaries for that Workflow, overtime, based on Build Number.  You can also view the Binary Results for your “Latest Run” of a Workflow by going to the “Latest Run” menu Option off of the “MyWorkflows” option on the Management Control Console.   You cannot view the binary results from inside the Eclipse RCP.
