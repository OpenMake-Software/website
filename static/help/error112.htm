ERROR 11:

          Could not change to the directory specified with the -c option.

 

Description:

The -c option was passed to bldmake. The -c option changes the directory where bldmake will execute.  The -c option requires a parameter that is the name of the directory where bldmake will execute.  This error displays when the directory is not available or cannot be written to.

 

Resolution:

Make sure the directory exists or that it is not a protected, read only directory.  You should also make sure that there is space available in the directory.
