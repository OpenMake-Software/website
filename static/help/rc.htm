-rc

Retrieve Configuration.  The contents of the file specified by this parameter are passed to the Retrieve Program (-rp) as parameters. This can be used provide a list of configuration information needed by the retrieve program such as log in ID, database name, etc.  OOMLOM locates the configuration file by searching the Search Path.

OOMLOM –rp <script name> –rc <configuration file name>

example:

OOMLOM –rp getfiles.bat –rc configfile.txt

<configuration file>
