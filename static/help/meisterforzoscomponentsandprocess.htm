Meister for z/OS Components and Process

 

Meister for z/OS provides support for the z/OS operating system running on the IBM mainframe platform.  It provides the ability to execute builds on the z/OS platform and to maintain build information through native ISPF panels.  Meister for z/OS features include Impact Analysis reporting, Footprinting, Bill of Material Reports and Smart Logs for builds that run natively on the mainframe z/OS platform.
