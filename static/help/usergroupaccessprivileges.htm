User Group Access Privileges

 

The default User Group has read-only access to the Administrator functions and has full access to all items under the "Developer" views.    These views include:

 

•  Target Explorer - Allows Developers to setup builds and define Targets to be included in the Projects.

•  Workflow Explorer - Allows customized Workflows to be defined and executed.

•  Report Explorer - Displays Logs, Impact Analysis, and Metrics Reports from previous builds.

•  Preferences – Allows Users to customize the Machine settings for their local installation of the Client.

•  Help - Displays field level help.

 

*Note:  If you are only using the free Meister’s CI functionality you will only have access to Workflows and Report Explorer.

 

Users have no access to the Administrator functions and have limited access to standard User Views.  These Views include:

 

•  Workflow Explorer - Allows customized Workflows to be defined and executed.

•  Report Explorer - Displays Logs, Performance and Metrics from previous Workflows.

•  Preferences – Allows Users to customize the Machine settings for their local installation of the Client.
