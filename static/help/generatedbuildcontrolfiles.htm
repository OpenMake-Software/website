Generated Build Control Files

 

OOMLBLDM generates a Build Control file by using the data stored in the Knowledge Base Server.  This information includes the Target Definition (based on Project), the Search Path, the Build Types, the Build Rules and the Scripts.  The batch program, OOMLBLDM uses this information to create the Build Control files.  OOMLBLDM requires a minimum of two parameters, namely: the Project Name and the Search Path Name.  Using these parameters, OOMLBLDM searches for the Project Name in the data sets listed in the Search Path, looking for .tgt files that belong to the Project. From the Target Definitions, OOMLBLDM  then finds the corresponding Build Types, Rules and Scripts.  OOMLBLDM follows the Build Rules, in order to generate the Build Control file for the Targets.
