Workflow Detail Screen

 

Workflow detail can be viewed by opening the Workflow and displaying the Workflow Detail screen. Workflows can be customized to perform pre and post actions.  Workflows are organized into private builds (listed under "My Workflows") or "Public Workflows".   Workflows can be "promoted" from Private to Public by the User who created the Workflow.  The creator of the Workflow can "demote" a Public Workflow back to a Workflow.  An Administrator, or User who is a part of a group with correct privileges can "demote" any Public Workflow to a Private Workflow.
