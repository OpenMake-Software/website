-rp

Retrieve Program.  This parameter is used to tell OOMLOM to execute a script that will retrieve files from a version control tool. The file specified by this parameter will be executed before any dependency checking has occurred by OOMLOM.  OOMLOM locates the script file to execute by searching the Search Path.

OOMLOM –rp <script name>

example:

OOMLOM –rp getfiles.bat

<retrieve script file name>
