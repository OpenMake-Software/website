Navigating the UI

 

In the Explorers, such as the Group Explorer, User Explorer, Workflow Services Explorer, Deploy Services Explorer, Reports Explorer and Logs Explorer, the Refresh button can be used to reload the data from the Knowledge Base Server into the Explorer.  When a refresh is used and an Editor is opened, the Explorer will be updated with the new data, but the Editor will still contain the old data.  If the Editor is closed and reopened it will then contain the new data from the Knowledge Base Server.
