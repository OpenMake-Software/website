CheckStyle

CheckStyle is a development tool to help programmers write Java code that adheres to a coding standard.  This Workflow Activity allows you to call CheckStyle as part of your standard Workflow.

 

The source code directory is passed in as a parameter

The source to include is passed in as a parameter

The source to exclude is passed in as an optional parameter

 

Install Instructions:

1.   Download the latest version of Checkstyle from sourceforce

      - https://sourceforge.net/project/showfiles.php?group_id=29721

2.   Select the Download link and on the page that’s displayed, click to download the checkstyle zip file

3.   Unzip the file to a local directory on your hard drive

4.   Once unzipped locate the jar files (files should be in the first folder of the unzipped file)

5.   Copy all the jar files from the step above to <MEISTER_INSTALL_PATH>/client/tools/checksytle/lib
