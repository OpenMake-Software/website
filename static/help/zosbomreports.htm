z/OS BOM Reports

 

To create a Build Audit Report for a build using the ISPF panels, modify the OOMLOM options by selecting a “/” next to the “_ / to choose” field.   Select the “Create Build Audit Report” option by entering a “/”.  Execute your build.

 

All Build Audit Reports generated through the ISPF panels are saved to:

<user’s high level qualifier>.BOM.DATA

One Build Audit Report is saved for each Target in the build.

 

To generate a Build Audit Report using batch JCL, enter  “-b <bom file name>” option on the OOMLOM parameters line.  Execute the JCL.  The Build Audit Report will be saved to the location passed with the –b flag to OOMLOM.
