FindFlag( $flagString, @possibleFlags )

 

This function verifies the existence of @possibleFlags within a $flagstring. The first flag matched is returned.

 

USAGE:

 $myFlag = FindFlag( $flagString, @possibleFlags );

 

Example:

 

@possibleFlags = qw( javac jikes jvc );

$flagString = 'javac -deprecation -g';

$compiler = FindFlag( $flagString, @possibleFlags );

 

RETURNS:

First found item in the string from the possible flags list.
