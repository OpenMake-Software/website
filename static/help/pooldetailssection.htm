Pool Details Section

 

This section shows you the initial Workflow directory from which the Workflow will be executed for each Server Pool defined in your Workflow Hierarchy.  You will need to define the directory where the Workflow execution will be initiated in the “Initial Workflow Directory” field.  The directory which you enter in this field will be created on the machine selected by the Server pool.  Each Workflow Activity in the Workflow will use this machine directory unless it is overridden at the Workflow Activity level. You can have a uniquely defined “Activity” directory should you want a particular Workflow Activity to be executed in a different location.  Otherwise the Workflow Activity will execute using the “Initial Workflow Directory.”
