Weblogic Build Methods

 

Build Method Name

Description

Weblogic APPC Ant Task.sc

Executes the Weblogic APPC compile using the Ant Task

Weblogic APPC Task.sc

Executes the Weblogic APPC compile using compiler directly

Weblogic Clientgen Task.sc

Generates a Web Service-specific client JAR file that client applications can use to invoke both WebLogic and non-WebLogic Web Services

Weblogic EJBC Task.sc

Generates and compiles EJB 2.0 and 1.1 container classes

Weblogic JSP Compile Task.sc

Compiles the JSP for the Weblogic Server

Weblogic Servicegen Task.sc

Creates all the needed Web Service components and packages them into a deployable EAR file
