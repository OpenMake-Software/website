Execute Settings Section

 

 

The Execute Settings Section displays all Server Pools that are being used as part of the Workflow, as defined through the Workflow Activities screen under the Activities Tab.  “MyComputer” will show as a default machine.  You can use the Execute Setting option to limit the steps that will run during a Workflow.  For example, if you have steps executing on a Linux Server Pool, and other steps executing on a Windows Server Pool, you can un-check the Linux machine, and only the Workflow Activities defined to the Windows Pool will execute.  The default selection will show all Server Pools in the “checked” state.  This means that all Workflow Activities will be executed as a default setting.
