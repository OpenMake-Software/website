Sun Solaris and Sybase Build Methods

 

Build Method Name

Description

Sun App Client Jar Task.sc

Creates a Jar file for an Application Client

Sun Ar Archiver.sc

Executes ar to create an archive library

Sun C and Cpp Compiler.sc

Executes the Sun CC or cc compiler to create objs

Sun C and Cpp Linker.sc

Executes the Sun CC or ld to create executables and shared libraries

Sun Curl Compiler.sc

Execute the curl cross compiler

Sun Esql Compiler.sc

Executes the esql compiler to load stored procedures, functions and DDL into an Informix database

Sun Uil Compiler.sc

Execute the user interface language compiler

Sun Url Compiler.sc

Execute the curl cross compiler

Sybase BCP.sc

Executes the SyBase Bulk Copy Program

Sybase ISQL Compile All.sc

Executes the isql to load stored procedures, functions and DDL into an Informix database
