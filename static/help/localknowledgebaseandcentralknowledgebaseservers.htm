Local Knowledgebase and Central Knowledgebase Servers

The Local Knowledgebase server is the repository for Workflow metadata information created by a single user. This private server contains the actual Workflows, Logs, Reports, and Metrics created by an individual user.

 

The central Knowledgebase server is the repository for Workflow metadata information for a community. This server contains the actual Workflows, Logs, Reports, and Metrics created by a community of Users.  When a group of developers use the Knowledgebase, Workflows can be shared across the team.  These Public Workflows are stored on the central Knowledgebase, along with all Workflow Logs, Bill of Material Reports, Metrics Report, testing results and static analysis.
