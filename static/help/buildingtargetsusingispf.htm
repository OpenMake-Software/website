Building Targets Using ISPF

 

Following is a description how to execute builds from the native mainframe ISPF panels.  These panels support only “Developer” functions.  No “Administrative” features are available through the IPSF panels.  In order to create, update, or delete Groups, Users, Projects, Search Paths, Build Types or Rules, you must use the Web Client and have the correct privileges.  ISPF panels allow for the adding, updating and deletion of Targets and the execution of builds for all Projects which you have privileges to access.
