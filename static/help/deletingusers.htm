Deleting Users

Users must be deleted by the Administrator via OM Admin Perspective. Table GU-5 will provide the simple steps.

 

Table GU-5  Deleting Users

Steps

Action

1

From the main menu select Windows – Open Perspective – OM Admin.

2

Select the User Explorer tab for a list of all Users

3

Select the User and right mouse click to display the Delete option.

4.

Select Delete

 

When a User or Administrator is deleted, their associated User Profiles and directories are automatically deleted from the Knowledge Base Server.
