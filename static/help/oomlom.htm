OOMLOM

 

OOMLOM is Meister for z/OS's main engine.  OOMLOM reads the Build Control file generated by OOMLBLDM and then executes the build process. During execution, OOMLOM scans the source code for dependency information determining which modules should be built. OOMLOM then generates standardized compile JCL and submits the JCL to the internal card reader for execution.  The results of the job submission are reported back to the Knowledgebase Server in the form of Build Logs.  In addition, if Impact Analysis is requested, the Impact Analysis report is written to the Knowledgebase Server.
