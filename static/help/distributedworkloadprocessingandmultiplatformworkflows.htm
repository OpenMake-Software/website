Distributed Workload Processing and Multi-platform Workflows

 

Remote Agents and Server Pools support the ability to distribute the workload of a Workflow across multiple machines with diverse operating systems.  A Remote Agent allows you to execute Workflows on a machine other than the one you are working from. Remote Agents are assigned to a Server Pool.  A Server Pool  must have a least one Remote Agent. Server Pools are defined to a Workflow, or at the individual Workflow Activity.  Server Pools improve processing speed as multiple Workflow Activities can be defined to execute simultaneously across multiple machines. 

 

Note: Remote Agents are upgrades to your standard installation and must be purchased as a separate license.
