Managing Users and Groups

 

Your Administrator has the ability to control the levels of access at both a User and Group level.  Privileges are assigned to Groups based on Build Services, Dependency Directories and Public Workflows. User and Group control is done through the OM Admin perspective or by using LDAP.   This guide explains the default Groups and how to manage Users and Groups using the OM Admin perspective and using LDAP.
